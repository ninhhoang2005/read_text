## readText

This is a very useful text reading aid for long files.

## Key Capabilities

the application focuses on user experience, offering the following functions:

## main feature

We have designed it to bring you the most efficiency when reading a text.

*   **Choose your reading voice**: Allows users to choose the reading voice according to their preferences.
*   **save audio**: Users can save text to read as audio.
*   **save text**: Another pretty cool solution is that the user can save the text entered in the cell into a plain text file.
*   **open text file**: You can read the entire long text by uploading it.
*   **Retrieve text from clipboard**: Get text from the user's clipboard.
*   **Listen text**: Listen again to the text in the input box.
